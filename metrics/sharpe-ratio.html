Sharpe Ratio

Home Information Ratio Volatility & Beta

Definition

Sharpe Ratio estimates how much excess return an investment earns for each unit of total risk.

Formula: (Portfolio Return − Risk-Free Rate) / Portfolio Volatility

Worked Example

Suppose:

  • Portfolio return = 12%
  • Risk-free rate = 3%
  • Portfolio volatility = 15%

Step 1: Excess return = 12% − 3% = 9%

Step 2: Sharpe = 9% / 15% = 0.60

Interpretation: the portfolio generated 0.60 units of excess return per 1 unit of total risk.

Quick Comparison Example

  • Portfolio A Sharpe = 0.60
  • Portfolio B Sharpe = 0.85

If measured over the same period and methodology, Portfolio B has better risk-adjusted performance.

Interpretation Notes

  • Higher is generally better.
  • Negative Sharpe means return was below the risk-free rate.
  • Use consistent time windows and volatility methods when comparing.
