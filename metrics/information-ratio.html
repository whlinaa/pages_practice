Information Ratio

Home Sharpe Ratio Volatility & Beta

Definition

Information Ratio evaluates active return versus a benchmark, adjusted by tracking error.

Formula: (Portfolio Return − Benchmark Return) / Tracking Error

Worked Example

Suppose:

  • Portfolio return = 11%
  • Benchmark return = 8%
  • Tracking error = 4%

Step 1: Active return = 11% − 8% = 3%

Step 2: Information Ratio = 3% / 4% = 0.75

Interpretation: the strategy produced 0.75 units of active return for each 1 unit of active risk.

Mini Monthly Example

Monthly active returns (%): 1.0, 0.4, -0.3, 0.8, 0.1, 0.6

  • Average active return (monthly) ≈ 0.43%
  • Tracking error (monthly std. dev.) ≈ 0.48%
  • Monthly Information Ratio ≈ 0.43 / 0.48 = 0.90

This indicates consistent outperformance relative to the benchmark during the sample period.

Why It Is Useful

  • Helps evaluate active managers.
  • Rewards consistency, not one-time outperformance.
  • Useful when benchmark choice is clear and relevant.
